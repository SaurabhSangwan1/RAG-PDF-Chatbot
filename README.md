# RAG PDF Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that allows users to interact with PDF documents using Large Language Models.

## Features

- PDF Upload
- Semantic Search
- Question Answering
- Context-Aware Responses
- Vector Database Integration

## Tech Stack

- Python
- LangChain
- OpenAI API
- FAISS
- Streamlit

## Project Status

Currently under development.

## Future Enhancements

- Multi-PDF Support
- Chat History
- Source Citations
- Local LLM Support
